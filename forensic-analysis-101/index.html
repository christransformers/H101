Core Forensic Topics

Every graduate completes evidence handling with valid forensic rigour, documentation, and structured reasoning chains.

Hacking101 Forensic Analysis 101 topics cover: legal foundations, forensic imaging, evidence preservation, artefact analysis, malware analysis, incident triage, and courtroom-ready forensic reporting — taught via live Australian-hosted lab environments.

Evidence Collection

Disks, phones, memory, cloud credentials. Collection according to current AFP and state forensic guidelines.

Live Response

Volatile evidence capture, snapshotting, memory acquisition, and ensuring chain-of-custody is not broken.

Disk Analysis

File system artefact recovery, timeline buildout, slack space, unallocated clusters, deleted artefacts.

Malware Forensics

Static and dynamic analysis, unpacking, memory-resident payload reconstruction, and IOCs.

Document / Handwriting Forensics

Questioned documents, signature verification, forger trait identification, and report construction.

Court-Ready Reporting

Structuring evidence summaries, technical statements, and expert witness testimony preparation.
